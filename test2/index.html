Project Description

Project Description

For this project, I am planning to create a cyberpunk-themed webpage that includes a very simple JavaScript login authentication form. To go with this form, I want to create a tutorial for visitors on how to potentially use simple website exploitation techniques to potentially crack or bypass the password authentication required by the login form.

As a neat add-on, I am also hoping to make the site double as a directory on current trends in cybersecurity, particularly relating to cyber threat intelligence. Unfortunately, I've read that JavaScript can't parse and pull data from external sites to display them (I believe I would have to use a programming/scripting language such as python for that, and I am unsure whether or not this can fit with the scope of my project).

If for some reason I am unable to describe how to break the authentication form, I can also focus primarily on creating my library of threat intelligence feeds, and using JavaScript instead to animate icons for each directory selecrtion.

Rationale

My rationale for building this project.

I am an aspiring cyber security professional who wants to specialize in cyber Threat Intelligence gathering. I think it would be cool to share my interest ii the topic in a cool and somewhat interactive way with my peers, professors, and other aspiring cyber professionals in a stylish way that I can identify with. The cybersecurity industry is suffering from a significant talent and skills shortage, and this website can be a potentially way to spark new interest in potential security candidates.

Target audience

My target audience for this proposed site.

My intended target audience for this proposed site is anyone who wants to learn something new about cybersecurity, and also for cyber security practitioners and personnel who want to get the latest scoop on threat intelligence. But I hope that this can especially appeal to lovers of the cyberpunk genre and general tech enthusiasts such as myself.

Javascript Incorporation

Authentication!

I plan to leverage HTML and CSS to create a sort of cyberpunk ambiance. I would prefer to select a color pallette that matches with those of cyberpunk artwork, such as that found on the BladeRunner or Neuromancer cover art. I can also decorate the site with more retrowave pictures or .gif files.

The implementation of JavaScript is critical for this project in the creation of a simple login form, as well as to store variables that can be inputted for a successful login. I have found multiple ways of creating an authentication form online (all achieving basically the same thing), and one particular way includes using arrays and variables to test for for what inputs in the login form return a true boolean value. So in this way, no databases (such as SQL are required).

Again, JavaScript can also be used to create animations for icons and directory selections when a user moves their mouse over an object on the site
